## tutorial
1. download it [here](https://github.com/KhanhNguyen9872/Convert_Marshal-PYC/archive/refs/heads/main.zip) or git clone it!
2. unzip it!
3. run 'cv_marshal_pyc-py<version>.py'
4. input your marshal or pyc file!
5. done!
